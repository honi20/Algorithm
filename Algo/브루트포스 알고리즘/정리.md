# 브루트포스 알고리즘
* brute: 무식한 / force: 힘
* 완전탐색 알고리즘
* 순차탐색, dfs, bfs의 기본적 도구
<br><br>

>## 방법
1. 다 해보는 코드 일일이 작성
2. 반복문 사용
3. 순열 사용
4. 재귀 호출 사용
5. 비트마스크 사용
<br><br>

>## 시간복잡도  
* O(경우의 수 x 방법 1개를 시도해보는데 걸리는 시간복잡도)
